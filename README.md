# Oregonator - Modelo Matemático da Reação de Belousov-Zhabotinsky
Trabalho final para disciplina de Equações Diferenciais que tem como objetivo investigar o Oregonator, um modelo matemático desenvolvido para descrever o comportamento oscilatório da famosa reação química de Belousov-Zhabotinsky (BZ).

A reação de Belousov-Zhabotinsky é conhecida por seu comportamento oscilatório autocatalítico, desenvolvida inicialmente por Belousov com a finalidade de encontrar um análogo não-orgânico para a química observada no Ciclo do Ácido Cítrico (também conhecido por Ciclo de Krebs). Modelada pela primeira vez por Field, Köros e Noyes no modelo conhecido por FKN, a reação foi então simplificada para o modelo Oregonator por Field e Noyes. Esse modelo sintetizou as 10 reações iniciais para 5 versões generalizadas e apresentou a modelagem da concentração dos reagentes oscilatórios em um sistema de 3 Equações Diferenciais Ordinárias não-lineares. Esse sistema foi explorado nesse repositório computacionalmente, através de soluções pelos métodos de Euler e Range-Kutta, e semi-analiticamente, baseados no trabalho de Misra, I. e Ramanathan, V. (2022).

Para melhor entendimento dos objetivos e reultados obtidos recomenda-se a leitura do trabalho escrito "Projeto_Final_Equaçoes_Dif_Oregonator" em formato PDF. Em anexo segue também o arquivo "Calculos_Semi_Analitica" contendo todo o processo de cálculo para o desenvolvimento de uma solução semi-analítica.
